Para compilar los .class
    Con el pom:
        mvn install

    Con javaC:
        javac --module-path /usr/lib/javafx-sdk-21.0.5/lib/ --add-modules javafx.controls,javafx.fxml -d bin $(find src/main -name "*.java")

Para ejecutar

    Servidor:
        Con el pom:
            java -cp target/classes/ mx.unam.ciencias.modelado.proyecto2.proxy.ClienteRemoto
        
    Como Usuario.
        Con el pom:
            mvn javafx:run

        Con javaC:
            java --module-path /usr/lib/javafx-sdk-21.0.5/lib/ --add-modules javafx.controls,javafx.fxml -cp bin mx.unam.ciencias.modelado.proyecto2.Main
